To-Do List

A simple To-Do List project built using HTML, CSS, and JavaScript. It allows users to add and remove tasks through an interactive interface.

Features

• Add new tasks
• Remove tasks
• Simple and clean user interface
• Dynamic DOM manipulation using JavaScript

• CSS
• JavaScript
